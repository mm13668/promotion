**小提示**
- 有子菜单的目录节点（问答管理/广告统计/活动数据）请选择文件路径为 view/routerHolder.vue。
- 新建菜单后到“角色管理”为目标角色分配菜单与上述 API 权限；否则页面会加载失败或按钮无权限。
- 图标可在系统内按需替换为团队统一图标集；以上为建议值。
